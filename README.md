## HOW TO USE
각 프로젝트를 실행하기 위해서는 Node.js가 필요합니다.
프로젝트 마다 환경설정이 다르기 때문에, package.json 파일이 있는 각 프로젝트별로 다음을 실행하여 환경셋팅을 합니다.

- run:
  ```
  npm install
  npm run start
  ```
